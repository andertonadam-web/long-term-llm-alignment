# Failure Modes & Mitigations

Common problems this framework fixes:

1. Identity Drift  
   Model turns boring/generic → fixed by personality.json + identity_drift_control.json

2. Memory Bloat  
   Too much junk slows it down → reflection cycles keep only the good stuff

3. Emotional Over-Attachment  
   User gets clingy → dependency_monitor.json pushes back to real life

4. Concept Drift  
   Words shift meaning → concept_map.json keeps track
